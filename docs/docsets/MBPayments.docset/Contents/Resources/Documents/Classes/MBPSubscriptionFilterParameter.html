MBPSubscriptionFilterParameter

@interface MBPSubscriptionFilterParameter : NSObject

Filter that can be used to filter only the sections that belongs to users with an active/inactive subscription.

  • onlyActive

    If true this filter will return the sections that belongs to users with an active subscription. Otherwise it will return the sections that belongs to users with an inactive subscription. Default value is TRUE.
